# StarryNight

* Unity로 만든 Android용  조합 클리커 게임입니다.
* 클리커를 이용하여 아이템을 모아 조합하여 도감을 모으고 퀘스트를 수행하면서 시나리오를 끝까지 진행하는 것이 목표입니다.
* Raycast를 이용한 아이템 조합, 데이터 파일 입출력를 이용한 로컬 아이템 및 서적 DB 생성 및 이를 이용한 도감 페이지 제작, 튜토리얼 기능을 작업했습니다.
